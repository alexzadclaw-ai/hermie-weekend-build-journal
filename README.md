# Hermie Weekend build journal

Anonymous first-person build log for the journey from a first AI-agent conversation to a live local media brand: website, Telegram channel, Cloudflare Pages deployment, social previews, Lighthouse fixes, and the early monetization thesis.

This repo is intentionally Markdown-first so it can later become an Astro, Hugo, Jekyll, Next.js, or other static blog.

## Editorial choices

- Voice: anonymous first person
- Real name: omitted
- Secrets, tokens, OAuth URLs, private chat IDs: omitted
- Affiliate links: not included yet
- Format: Markdown with frontmatter
- Status: drafts
